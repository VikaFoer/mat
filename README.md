# Веб-інтерфейс для вивчення дискретної математики

Цей проект містить веб-інтерфейс для вивчення дискретної математики з організацією матеріалів по вкладках.

## Структура проекту

- `index.html` - головна HTML сторінка з вкладками для кожної лекції
- `styles.css` - стилі для красивого та сучасного інтерфейсу
- `script.js` - JavaScript для перемикання вкладок та додавання контенту
- `README.md` - документація проекту

## Як додати матеріали з PDF

### Варіант 1: Додавання PDF файлу (iframe)

Після того, як ви завантажите PDF файл у папку проекту, використовуйте функцію `addPDFViewer()`:

```javascript
addPDFViewer('lecture1', 'path/to/your/file.pdf');
```

### Варіант 2: Додавання контенту з PDF (HTML)

Якщо ви конвертували PDF в HTML або маєте текстовий контент, використовуйте функцію `addPDFContent()`:

```javascript
addPDFContent('lecture1', '<h3>Заголовок</h3><p>Текст матеріалу...</p>');
```

## Запуск

Просто відкрийте `index.html` у веб-браузері. Для кращої роботи з PDF файлами рекомендується використовувати локальний веб-сервер.

### Запуск локального сервера (опціонально)

```bash
# Python 3
python -m http.server 8000

# Node.js (якщо встановлено http-server)
npx http-server -p 8000
```

Потім відкрийте `http://localhost:8000` у браузері.
