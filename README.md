# Веб-інтерфейс для вивчення дискретної математики

Цей проект містить веб-інтерфейс для вивчення дискретної математики з організацією матеріалів по лекціях.

## Структура проекту

- `index.html` - головна HTML сторінка з sidebar навігацією
- `styles.css` - стилі для красивого та сучасного інтерфейсу
- `script.js` - JavaScript для навігації та завантаження зображень
- `README.md` - документація проекту
- `1 лекція/` - папка зі зображеннями для першої лекції

## Структура навігації

Сайт має sidebar зліва зі списком лекцій:

## Як додати матеріали

### Для першої лекції:
Зображення вже налаштовані з папки `1 лекція/`. Вони автоматично завантажуються при відкритті лекції.

### Для інших лекцій:
1. Створіть папку з назвою `N лекція` (де N - номер лекції)
2. Додайте зображення в цю папку
3. Оновіть об'єкт `lectureImages` в `script.js`, додавши масив шляхів до зображень

Приклад:
```javascript
lectureImages[2] = [
    '2 лекція/image1.png',
    '2 лекція/image2.png',
    // ...
];
```

## Запуск

Просто відкрийте `index.html` у веб-браузері. Для кращої роботи рекомендується використовувати локальний веб-сервер.

### Запуск локального сервера (опціонально)

```bash
# Python 3
python -m http.server 8000

# Node.js (якщо встановлено http-server)
npx http-server -p 8000
```

Потім відкрийте `http://localhost:8000` у браузері.
